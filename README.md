ROS Kinetec, Gazebo 7, Ubuntu 16.04 


**Install script:**
``` 
>> CTRL+ALT+T
$ sudo apt-get install dialog
$ cd shfiles
$ chmod a+x -v *.sh
$ cd
$ gedit .bashrc
>> PASTE: export PATH=$PATH:~/shfiles
>> SAVE
>> CTRL+H
>> CUT: .dialogrc
>> PASTE: /home
```

**Run script:**
``` 
$ cd shfiles
$ execpepper.sh
```
