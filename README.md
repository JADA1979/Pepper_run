ROS Kinetec, Gazebo 7, Ubuntu 16.04 


**Install script:**
``` 
CTRL+ALT+T
$ sudo apt-get install dialog
$ cd shfiles
$ chmod a+x -v *.sh
$ cd
$ gedit .bashrc
PASTE and SAVE: export PATH=$PATH:~/shfiles
```

**Run script:**
``` 
$ cd shfiles
$ execpepper.sh
```
